# SCP 명령어 파일 전송

### 1. 다른 서버의 파일 가져오기

현재 접속한 서버에서 아래 명령어를 통해 다른 서버에 있는 파일을 가져오는 방법

```bash
// -r : 하위 폴더까지 같이 복사  
scp -r [다른서버ID]@[다른서버IP]:[다른서버파일/폴더경로] [현재서버 복사할 경로]
```

### 2. 현재 서버의 파일 보내기

현재 접속한 서버에서 아래 명령어를 통해 파일을 다른 서버로 보내는 방법

```bash
// -r : 하위 폴더까지 같이 복사  
scp -r [현재서버 복사할 경로/파일] [다른서버ID]@[다른서버IP]:[다른서버파일/폴더경로]
```
